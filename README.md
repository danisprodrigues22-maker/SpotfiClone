# LIONS MUSIC – Backend

Backend da aplicação LIONS MUSIC, um Spotify Clone desenvolvido como projeto acadêmico.

## Tecnologias

- Node.js
- Express
- MongoDB
- Mongoose
- JWT Authentication

## Requisitos

- Node.js 18+
- MongoDB rodando localmente

## Instalação

Clone o repositório:

git clone <URL_DO_REPOSITORIO>

Entre na pasta do backend:

cd backend

Instale as dependências:

npm install

Crie o arquivo de variáveis de ambiente:

cp .env.example .env

Edite o arquivo `.env` se necessário.

## Rodando o servidor

npm run dev

ou

npm start

O servidor iniciará em:

http://localhost:4200
